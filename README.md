## 简介
CME项目旨在利用数据搭建一套自动预测太阳风到达时间的系统。其包含主流机器学习方法SVM、RF、Adaboost、XGboost等。
同时为进一步增强模型的抗干扰能力，我们在训练过程中加入了随机噪声以及扰动点以增强模型鲁棒性。
